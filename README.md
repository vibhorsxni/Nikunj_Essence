# Nikunj Essence - Premium 3D Attar Experience

Welcome to the **Nikunj Essence** digital storefront—a highly polished, interactive 3D web experience designed to showcase authentic, natural Indian Attars and Floral Waters.

## ✨ Features
- **Interactive 3D Product Showcase:** Experience premium glass attar bottles in 3D right in your browser, powered by React Three Fiber.
- **Luxurious Aesthetic:** A custom dark green (`#0A1C12`) and gold (`#D4AF37`) color palette that reflects the brand's heritage and premium nature.
- **Mobile-First Design:** Fully responsive layouts, swipeable product cards, and floating CTA buttons designed specifically for mobile smartphone users.
- **WhatsApp Checkout:** A seamless, friction-free checkout system that instantly formats the customer's cart into a readable order summary and redirects directly to the owner's WhatsApp Business.
- **Fluid Micro-Animations:** Beautiful page transitions, floating gold particles, and glassmorphism UI elements built with Framer Motion.

## 🛠️ Technology Stack
- **Framework:** React 18 & Vite
- **Styling:** Tailwind CSS v3
- **3D Engine:** Three.js & React Three Fiber (`@react-three/fiber`, `@react-three/drei`)
- **Animations:** Framer Motion
- **Icons:** Lucide React

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (v18+ recommended) installed.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/vibhorsxni/Nikunj_Essence.git
   cd Nikunj_Essence
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the local development server:
   ```bash
   npm run dev
   ```

### Deployment
This project is configured to automatically deploy to **GitHub Pages** via GitHub Actions.
Any code pushed to the `main` branch will instantly trigger a deployment!
